Javascript

Javascript wurde 1995 von Netscape entwickelt und war dafür gedacht um dynamischeres HTML produzieren zu können.
Obwohl es dabei Namensgleich mit Java ist, hat es dennoch nicht sehr viel damit gemeinsam und unterscheidet sich sogar grundlegend von diesem.

"Um die Popularität von Java zu nutzen, wurde LiveScript in JavaScript umbenannt, obwohl die beiden Sprachen voneinander unabhängig entwickelt wurden und völlig unterschiedliche Grundkonzepte aufweisen."

Javascript hat dabei keinen Compiler und funktioniert über einen im Browser integrierten Interpreter. Den Sprachkern, der übersetzten Syntax macht dabei ECMAScript aus.
Die Funktionen von Javascript sind dabei sehr umfangreich. Es erlaubt vor allem das Ausführen von Gleichung, besitzt unterschiedliche Datentypen(Strings, Integers, Variablen usw) und kann z.B. auch Content von anderen Websites "fetchen". Dabei kann es sowohl in eine funktionale als auch objektorientierte Richtung programmiert werden.

Jit konvertiert Javascript code während während es ausgeführt wird.

Fingerprinting mit JS

Javascript kann auch dafür genutzt werden um über den HTTP Header informationen wie Browser Einstellungen, Betriebssystem, allg. Systeminformationen und mehr abgefragt und an den Server geschickt werden.
Diese Informationen können erneut abgefragt und dazu genutzt werden um individuelle Nutzer im Internet zu deanonymisieren. Auch ohne Cookies.

Integration in HTML und DOM

Ähnlich wie CSS kann Javascript auch in HTML integriert genutzt werden. Für kleinere Anwendungen reicht es aus das <script> Element zu nutzen. Dazu kommen dann noch die DOM Elemente welche es leicht ermöglichen, Elemente dynamiken zu verleihen.
Es können mit "document.GetElementById()" z.B. Elemente aufgegriffen und verändert werden.
